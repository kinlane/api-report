Extensis today released new updates to Portfolio 2016, the digital asset management (DAM) system designed to help companies and workgroups of all sizes efficiently manage asset creation, organization and utilization.

For companies that need to share files outside of their team or organization, Portfolio 2016 offers a module – Portfolio NetPublish – which makes it quick and easy for anyone to build sites for securely sharing assets online.

With today’s updates, Portfolio NetPublish sites can be highly customized to match the company’s brand, presenting digital assets from a portal with a customized look and feel. Customers can build custom NetPublish themes and assign them to any site on the server. Simply build it once, and anyone can use it.

New updates in Portfolio NetPublish build on the rich set of functionality Portfolio 2016 is known for, including:

  • Dependable version control & revision tracking
  • Asset check-out and check-In
  • File locking
  • Secure, centralized storage
  • Predictive, Google®-like search
  • Powerful metadata & keyword tagging
  • Convenient automatic cataloging
  • Systems integration with Portfolio’s RESTful API
  • Mobile upload from any iOS or Android™ device using Portfolio Flow®
  • Built-in backup and restore with a single click
  • Flexible user management with permission-based roles
  • Built-in support for Active Directory / Open Directory
  • Optional Single Sign-On with LDAP integration
  • On-Premise or cloud-based hosting

About Extensis’ DAM Solution: Portfolio
With the explosion of digital assets and media over the past decade, companies across the globe are facing an increasing issue with how to keep them organized, accessible and everyone using the right versions. Inefficiencies are growing, with companies losing time and money trying to locate files. Portfolio ensures users have immediate access to the correct files in the desired format in a central location. It is the only digital asset management solution that is affordable, simple to use, can be setup within a day, and provides everything organizations need to get started right out of the box. As a standards-based platform that works with other systems and existing workflows, Portfolio provides a single place to for teams to locate important files with consistent metadata information using a web browser or Mac and Windows desktop applications. Portfolio is ideal for anyone with large collections of images, videos, and marketing collateral.
