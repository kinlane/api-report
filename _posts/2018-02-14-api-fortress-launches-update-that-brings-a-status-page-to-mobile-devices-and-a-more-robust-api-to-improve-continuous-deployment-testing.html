API FORTRESS ANNOUNCES MOBILE STATUS PAGE, EXECUTE GROUPS OF TESTS BY API, AND A NEW TEST DASHBOARD

Update brings a status page for mobile devices, and a more robust API to help make part of a continuous deployment.

New York, NY - February 14, 2017 - API Fortress, a leading platform for API health, today announced updates that further the market leadership in API health. With a focus on flexibility in test execution, a UI update facilitates the process of launching tests manually. Also, an API update allows for the launching of groups of tests. This improves integration into existing continuous deployment setups using Jenkins, TravisCI, etc…

A feature, unique in the testing space, is the ability to login to API Fortress and get a completely mobile friendly view. Get a snapshot of the API’s health (accuracy and functional uptime), without needing to pinch and expand your way to the right place. It’s 2017, we know you’re on a mobile device, so let’s make life easier.

Free trials are available with full functionality on www.apifortress.com.
