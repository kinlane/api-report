Unicon, Inc., a leading IT consulting, services, and support provider specializing in open source for the education technology market, today announced the availability of Test Drive Open Analytics, a test environment designed for developers to experiment and capture learning events with OpenLRS. The test drive site provides each developer with their own instance of OpenLRS (an open source learning record store), in which learning events are captured in either IMS Caliper or xAPI format. Test Drive Open Analytics allows developers to experiment with open analytics with no upfront investment, ideally reducing the overall time to adoption of components like OpenLRS. Test Drive Open Analytics is a free to use cloud-hosted, multi-tenant, open learning analytics environment.

OpenLRS is an open source, secure, standards-based, standalone LRS that was built to fill the need for a storage mechanism for an open learning analytics environment. Built on a scalable architecture, using modern web technologies, OpenLRS provides the fast reads and writes necessary for a dynamic analytics environment.

OpenLRS is an open source project backed by the Apereo Foundation. As a commercial affiliate of Apereo, Unicon has a vested interest in ensuring the long-term success of OpenLRS and the Apereo Learning Analytics Initiative. The roadmap for Test Drive Open Analytics includes the addition of Apereo projects OpenDashboard and the Learning Analytics Processor (LAP).
