Pearson has made the stories from the Financial Times via an API available for over two years now, allowing developers to build applications around the financial and business news content--even making freely available for students.

According to Pearson, they have "offered its FT Press API for public use and exploration but it has become clear that its continued offering does not align to strategic goals of our new unified Pearson API platform.”

With this announcement Pearson has announced the deprecation of the public FTP Press API which is part of larger rmigration at Pearson to a single API strategy across the entire company, its products and partners.
