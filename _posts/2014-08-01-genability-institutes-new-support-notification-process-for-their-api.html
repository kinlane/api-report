The energy data API platform Genability is "instituting a new support notification process" to ensure API developers "receive timely notification of issues, outages and status of the Genability API and web applications”.

Genability will update API developers with:

  • Notification When API Outages Occur
  • Periodic Updates During API Outages
  • Notification When Things Are Back Normal
  • A Post-Mortem When They Know What Happened

You can receive notifications via email, and Genability separates issues by API and application, helping developers prioritize, based upon the individual issue.

A proactive API issue notification system is something more APIs are implementing to help support developers, and strengthen the feedback loop with API consumers.
