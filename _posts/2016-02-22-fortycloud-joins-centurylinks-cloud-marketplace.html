This integration allows customers of the CenturyLink Cloud platform to take advantage of FortyCloud’s offering that bundles all core network security components into a single, integrated product delivered as Software-as-a-Service. By leveraging FortyCloud’s comprehensive, end-to-end solution, CenturyLink Cloud customers can secure their cloud deployments in a fast, comprehensive way.

FortyCloud's Cloud Infrastructure Security Broker (CISB) solution comprises four layers:

  • Secure network layer provided by the FortyCloud software elements including firewall, identity and access management, VPN connectivity and more.
  • Abstraction layer unifying multiple cloud platforms to allow single policy language and ubiquitous network security capabilities.
  • Automation and orchestration layer providing flow-through configuration of the entire cloud deployment using policies and APIs as well as auto-discovery and auto-configuration of cloud resources.
  • Network visibility layer providing real-time view of the cloud deployment, collection of events, alerts generation and integration with external SIEM (security information and event management) and IDS (intrusion detection system) systems.

The CenturyLink Cloud Marketplace Provider Program allows participating technology companies, like FortyCloud, to integrate with the CenturyLink Cloud platform. These additional business-ready solutions are available to CenturyLink’s cloud, hosting and network customers.

“Joining the CenturyLink Cloud Marketplace was a priority for us,” said Amit Cohen, CEO of FortyCloud. “As a Cloud Infrastructure Security Broker, FortyCloud's primary goal is to enable enterprises with the choice of using multiple cloud platforms while remaining in compliance with their IT security requirements. With the recent addition of CenturyLink Cloud, FortyCloud can offer its services over multiple leading cloud platforms.”
“Virtualized network functions are becoming one of the key elements of any modern cloud deployment. FortyCloud’s Cloud Infrastructure Security Broker technology is ideally suited to many cloud network security requirements we see our customers needing,” said David Shacochis, vice president of platform enablement at CenturyLink. “We’re excited about the value FortyCloud can bring to the CenturyLink Cloud Marketplace Provider Program, which makes it easy for our customers to try new technologies in the network virtualization space.”

To start using FortyCloud on CenturyLink Cloud today, refer to the “Getting Started” guide on the CenturyLink Cloud Knowledge Base.

About FortyCloud
FortyCloud is a pioneer in cloud security, offering the industry's most comprehensive network and access security solution for public and hybrid cloud infrastructure deployments. The company's innovative solution incorporates Software Defined Security (SDS) and Software Defined Networking (SDN) to enable enterprises deployed in the cloud to build their own secure virtual private network over any number of cloud data-centers and platforms. Its award winning SaaS solution was launched in 2014 and has been since widely deployed on all major cloud platforms by worldwide customers and partners.
