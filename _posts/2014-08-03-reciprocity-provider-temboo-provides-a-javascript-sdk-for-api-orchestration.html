API interoperability, automation, and reciprocity provider Temboo has released their official JavaScript SDK, which allows you to build orchestrations around the 100+ APIs that are available on the platform, using purely client side JavaScript.

Temboo provides four simple examples to demonstrate what is possible:

  • Display the current temperature
  • Send an email via Gmail
  • Upload a text file to Dropbox
  • Save form data to a Google Spreadsheet

The Temboo JavaScript SDK makes it possible for developers to build widgets, and other custom scripts that even non-developers can reverse engineer and put to use for themselves, as other languages like PHP, Python and Ruby require server side environments, which might be out of reach for some users.

The release from Temboo, continues the trend around web APIs making data, content, and other programmatic cloud resources available to anyone.
