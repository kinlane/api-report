Cloud storage platform Dropbox just rolled out some new Android based features, including the ability to export files to an external SD card, something that is heavily requested by Android users.

Exporting Dropbox files to an external SD cards provides an easy way to transfer information between Android devices, and allows users to have important files available anywhere, even when offline.

The export to external SD card feature is just one feature in a lost list of recent features from the cloud platform, encouraging developers to use the platform for more than just file storage, but also syncing between platforms and devices, as well as an application settings and database storage solution for applications that run in the cloud.
