Users of API Fortress, the only fully codeless API testing and monitoring platform, can now create an API test with a RAML specification file in seconds. API Fortress is offering free accounts so users can get started and experience the only drag and drop visual test composer available.

API Fortress has just launched a major new feature that allows users to create an API test based on the RAML API definition spec. RAML (RESTful API Modeling Language) is a project led by a group of contributors including Airware, Akana Software, AngularJS, API Science, Cisco, Intuit, MuleSoft, and VMware. API Fortress allows users to build an API test without writing any code, and incorporating the ability to automatically generate a test from a RAML creates an additional method of doing so.

API Fortress is currently offering free accounts to try the first visually guided API testing platform. It is the only tool that offers the ability to test an entire API without needing to write any code. For the first time a QA, Product Manager, or Project Manager can easily do what was once only possible by developers. All tests are logged, all errors are immediately reported, and all data is available on the dashboard for deeper analysis.

API Fortress is exhibiting at TechCrunch Disrupt in New York City from May 9th - 11th.
