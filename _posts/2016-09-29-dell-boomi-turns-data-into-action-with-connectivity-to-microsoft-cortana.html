Dell Boomi today announced connectivity support for Microsoft Cortana Intelligence Suite, Microsoft’s cloud-based big data, advanced analytics and machine learning platform for organizations to transform data into intelligent action. Utilizing Boomi connectors, customers can quickly and flexibly integrate data and applications between Cortana Intelligence Suite, part of the Microsoft Azure cloud platform, and virtually any combination of cloud or on-premises data sources. Dell Boomi’s extension of its market-leading AtomSphere integration platform as a service (iPaaS) for Cortana Intelligence Suite better equips organizations to make informed, data-driven decisions, while seamlessly transitioning to hybrid IT environments of cloud-based and on-premises applications.

Available now, Dell Boomi’s new prebuilt connectors to Azure SQL Data Warehouse, Azure SQL Database and Azure Blob Storage – key technologies of Cortana Intelligence Suite – empower organizations to build connectivity at a fraction of the time and cost of custom-coded integration.

The new connectors strengthen the Dell Boomi and Microsoft relationship and underscore a commitment to help joint customers maximize the value of their IT investments. These connectors add to the Dell Boomi portfolio of integration solutions for Microsoft technologies, including the Microsoft Dynamics business suite, Microsoft SQL Server database and Microsoft SharePoint collaboration and document management platform. Dell Boomi made the announcement at the sold-out Microsoft Ignite conference in Atlanta, Georgia, where Boomi is demonstrating the new solutions at booth 801.

“Dell Boomi is proud to work with Microsoft to help customers capitalize on data and advanced analytics for improved insights and decision-making, a critical ingredient in digital transformation,” said Chris McNabb, Dell Boomi CEO. “Dell Boomi’s cloud integration connectors for Microsoft Cortana Intelligence Suite position organizations to accelerate time-to-value and progress smoothly to a cloud-first IT and analytics environment.”

Barb Edson, general manager, Marketing, Cloud & Enterprise, Microsoft Corporation, said, “Connectivity across data sources is a vital consideration as organizations continue adopting Microsoft Cortana Intelligence Suite for next-generation cloud analytics. Dell Boomi’s flexible and powerful connectivity solutions are a welcome addition to Cortana Intelligence Suite for customers to transform data into intelligence.”

Several Dell Boomi partners and customers have already leveraged these new Boomi connectors to help customers transition to Azure.

Tallan, Boomi partner, Microsoft Gold Partner and leader in custom application development and data management, uses Boomi connectors for Azure Blob Storage in the Tallan T-Connect EDI Management Suite. “The combined power and flexibility of the Dell Boomi platform with Tallan’s T-Connect extensible framework means that solutions can scale up and out to support high data throughput needs while seamlessly adhering to ever-changing EDI requirements,” said Stan Kennedy, Practice Director Integrations at Tallan.

PCL Construction, a leader in buildings, civil infrastructure and heavy industrial markets, uses Dell Boomi connectors to Azure for cloud, mobility, data management and integration initiatives. “Dell Boomi is a vital component to simplifying and standardizing integration between the Microsoft Azure platform and PCL’s application landscape, driving business value through reducing time to market and complexity while simplifying our cost structure,” said PCL Chief Information Officer Mark Bryant. “PCL uses Dell Boomi’s Azure Connectors to support our Boomi integration and master data management processes that expose Azure’s capabilities. For example, simplified data storage and analytics across PCL’s applications and business functions.”
