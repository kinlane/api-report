Stormpath announced the stormpath-nginx integration, which abstracts token authentication and validation upstream to Nginx. This integration allows API developers to expose OAuth 2.0 at the gateway in front of web and mobile applications and API services.

Nginx is one of the most popular open-source web servers and load balancers, and the integration with Stormpath allows API developers to generate OAuth2.0 access tokens for API services. Much like Nginx's other benefits - load balancing, caching, and monitoring - abstracted token authentication reduces the complexity of API management and development.

Additionally, managing API access with Stormpath allows developers to leverage a shared user store, with authentication and authorization capabilities, across multiple services. The stormpath-nginx integration augments Stormpath's existing offerings for API authentication: API key management and Token Authentication.
