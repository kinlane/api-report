3scale, the leading API management platform for providers, today announced that it has integrated Stormpath user management into its popular plans. Stormpath is a complete and easy Identity API for developers building web, mobile, and API-driven applications, with everything from user registration to authorization and user management. The combined capabilities simplify adoption of the complicated OAuth authentication flow, and continue to make 3scale the best platform on which to build OAuth-powered APIs.

The Stormpath integration is a powerful tool for any API provider who needs identity management. Via the integration, and with the code accessible on Github, customers simply “copy and paste” code to use Stormpath and 3scale together. By providing complete identity management for 3scale customers, the partnership brings radical simplicity to modern app development. This Stormpath integration is available beginning today as a no-cost option with all 3scale plans.

“The integration with Stormpath is the latest step from 3scale in becoming the standard for sophisticated APIs,” said Steve Willmott, CEO of 3scale. “We know that, due to the complexity of today’s development, in-house options are less feasible. This new partnership will make it easier than ever for API providers to implement a complete identity layer.”

 “OAuth presents API developers with a complex challenge – it’s complicated and labor-intensive to implement and secure,” said Alex Salazar, Stormpath CEO. “This partnership between Stormpath and 3scale makes it easy for API providers to quickly deliver reliable APIs with best-in-class security, authentication and identity management."
