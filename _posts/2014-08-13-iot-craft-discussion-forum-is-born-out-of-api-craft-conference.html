If you are in the API industry you know about the API Craft Google Group started by Apigee. The forum is the go to place when you want to have conversations with the experts across the API industry, which has been so active it has spawned Meetup groups in multiple cities, as well as the annual open conference in Detroit, API Craft.

After this years API-Craft conference, Chris Wilson (@datamoor) held a popular session on "APIs for Things”, reflecting the growing importance of APIs when it comes to IoT.

In response, the API-Craft organizers have now launched IoT Craft, a forum for discussion of all things APIs as they apply to the Internet of Things. If IoT Craft is anything like API Craft, it will be the place you go to learn about not just the tech of APIs being applied to IoT, but also the business, security, and other hot button issues around connecting devices to the Internet using APIs.
