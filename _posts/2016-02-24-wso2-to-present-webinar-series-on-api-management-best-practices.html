Opportunities to increase efficiency, build customer loyalty, and generate new revenue streams are leading organizations to adopt API-driven business models. However, those models are only as effective as the underlying API management system. WSO2 will present six webinars on how IT professionals can use WSO2 API Manager 1.10 to apply API management best practices in designing, publishing, managing, exploring and consuming APIs.

The API management webinar series will run February 9 – March 24, 2016. To learn more, visit http://wso2.com/landing/api-management-webinar-series-2016.

February 9 - Introducing the All New RESTful API for API Management

The latest release of WSO2 API Manager features a new RESTful API that helps users to manage APIs and subscriptions. This webinar will examine how the API addresses the challenges of extending and integrating WSO2 API Manager with other platforms while providing a better and more stable API developer experience. The session will look at the API’s design principle, implementation, access control and security, and versioning strategy. Additionally, it will present a roadmap for the API and discuss how developers can get involved in its evolution.

Jointly presenting will be Joseph Fonseka, WSO2 senior technical lead, and Malintha Amarasinghe, WSO2 software engineer. The webinar will be held Tuesday, February 9, from 9:00 a.m. – 10:00 a.m. PST. Visit http://wso2.com/library/webinars/2016/02/introducing-the-all-new-restful-api-for-api-management for more information.

February 23 – Planning Your API Journey: The Lifecycle of an API

Effective API lifecycle management enables collaboration across businesses, developers, and IT operations, resulting in rapid development and deployment cycles with increasingly reliable, stable and available APIs. This webinar will explore how new features in WSO2 API Manager 1.10 provide out-of-the-box support for API lifecycle management and enable developers to extend this functionality. It will then discuss the default API lifecycle engaged with an API and how to extend it, as well as how to extend the API lifecycle as a custom lifecycle.

Jointly presenting will be Lalaji Sureshika, WSO2 associate technical lead, and Chamila Adhikarinayake, WSO2 software engineer. The webinar will be held Tuesday, February 23, from 9:00 a.m.–10:00 a.m. PST. Visit http://wso2.com/library/webinars/2016/02/planning-your-api-journey-the-lifecycle-of-an-api for more information.

March 1 – APIs for API Management: Consume and Develop Apps

IT organizations increasingly seek services that allow them to develop applications that comply with their API management requirements. The webinar will discuss how WSO2 API Manager 1.10 supports this demand by exposing a REST API that application developers can consume when building their own portals. Additionally, this session will explore how to:

  • Take advantage of REST APIs available for API management.
  • Register dynamic clients for API consumption.
  • Leverage OAuth scopes for role-based authorization.
  • Obtain an access token and invoke APIs.
  • Develop apps by consuming store and publisher APIs.

Jointly presenting will be Lakmali Baminiwatta, WSO2 senior software engineer, and Tharindu Dharmarathna, WSO2 associate software engineer. This webinar will be held Tuesday, March 1, from 9:00 a.m. – 10:00 a.m. PST. For more information, visit http://wso2.com/library/webinars/2016/03/apis-for-api-management-consume-and-develop-apps.

March 8 – How to Enable Monetization of Your API Ecosystem

Organizations have started realizing the value of their data and services and are now developing ways to monetize them. This webinar will examine how WSO2 API Manager 1.10 features and integration points can be used to build an API monetization strategy and generate maximum revenue through APIs. Additionally, it will present an API billing model and usage plans. Then the session will discuss how to:

  • Integrate the API platform with a billing engine.
  • Monitor API usage.
  • Handle access control and rate limiting.

Jointly presenting will be Nuwan Dias, WSO2 technical lead, and Rukshan Premathunga, WSO2 software engineer. This webinar will be held Tuesday, March 8, from 9:00 a.m. – 10:00 a.m. PST. Visit http://wso2.com/library/webinars/2016/03/how-to-enable-monetization-of-your-api-ecosystem for more information.

March 16 – Deploy Your APIs for High Availability and Scalability

Beyond the core functionality, scalability and high availability are two of the most significant requirements of an API management platform. This session will examine how WSO2 API Manager 1.10 helps to address these performance demands while building a cost-effective, future-proof infrastructure that enables complete control. It also will review how a suitable deployment pattern can be selected to match the business requirement, network traffic and budget. Additionally the webinar will:

  • Provide an introduction to WSO2 API Manager components.
  • Review scalable deployment patterns.
  • Discuss how to achieve high availability.
  • Examine how to customize deployments.

Jointly presenting will be Janaka Ranabahu, WSO2 associate technical lead, and Harsha Kumara, WSO2 software engineer. This webinar will be held Wednesday, March 16, from 9:00 a.m. – 10:00 a.m. PST. Visit http://wso2.com/library/webinars/2016/03/deploy-your-apis-for-high-availability-and-scalability to learn more.

March 24 – Building an API Platform for Digital Transformation

Digitization helps people find new and more effective ways to connect, collaborate and conduct business, making it critical to transform existing legacy systems to support these new concepts. This webinar will examine how the WSO2 API management platform—including WSO2 API Manager, WSO2 Enterprise Service Bus, and WSO2 Data Analytics Server, and other WSO2 products—provides a complete solution for overcoming the challenges of digital transformation. Among the topics discussed will be how to:

  • Manage APIs exposed by other systems.
  • Integrate and modernize legacy systems easily.
  • Get more insight into APIs through big data analytics.

Jointly presenting will be Sanjeewa Malalgoda, WSO2 associate technical lead, and Dhanuka Ranasinghe, WSO2 associate technical lead. This webinar will be held Thursday, March 24, from 9:00 a.m.–10:00 a.m. PST. Visit http://wso2.com/library/webinars/2016/03/building-an-api-platform-for-digital-transformation to learn more.
